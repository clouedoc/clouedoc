# IDE

VSCode is an IDE of choice. It integrates a ton of extensions and works nicely.

## Extensions and configuration

I lied. VSCode is, in fact, a text editor, not an IDE. However, your goal will be to add extensions so that you can do everything you'd like with VSCode.

Discovering extensions is the hard part. I am so sweet that I will create a list of addons for the aspiring VSCode Jedi to install and try out.

Here it is:

| Name         | Description                                                          |
| ------------ | -------------------------------------------------------------------- |
| Grammarly    | a Grammarly extension for VSCode. Useful to have clean `README.md`'s |
| Paste images | a helpful tool to paste pictures into Markdown files                 |
